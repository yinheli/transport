## 开发初始化

clone 项目后，做如下操作，初始化工程。

```bash
# 初始化虚拟环境
python3 -m venv --copies --clear .venv

# 激活虚拟环境
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# all done，可以愉快的开发和调试了 🎉
# 推荐开发工具： vscode、pycharm
```

## 打包
